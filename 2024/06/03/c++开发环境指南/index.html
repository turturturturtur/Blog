vscode配置c/c++开发环境指南

第一步，下载并安装vscode

打开VScode官网（传送门），选择对应系统的安装程序并且下载，默认安装即可。

第二步，配置编译器

打开编译器官网，下载并安装编译器。
本教程以安装MingGW的g++编译器为例，其他编译器操作步骤雷同。

下载对应编译器

打开MinGW官网，点击如图的链接进行下载。

安装编译器

安装时选择安装位置，为了后续方便，建议选在C盘。目录不建议含有中文。记住这个目录，很重要！

配置环境变量

安装完成之后我们打开刚刚的安装路径，找到并打开MinGW -> bin,进入bin文件夹之后点一下这里，右键复制路径。注意看图，是MinGW下的bin文件夹，不是其他子目录下的bin文件夹！

再后，搜索环境变量。搜索环境变量

环境变量 -> 用户变量 -> Path ，双击打开Path变量，添加一项，内容填写为bin文件夹的路径。

最后，按下win+R并输入cmd，回车。在控制台中输入g++ -v，若有显示版本号，则证明配置成功。

第三步，配置vscode插件

汉化插件

进入vscode，按下Ctrl+Shift+X，搜索Chinese，选择安装Chinese（Simplified）则安装中文简体的语言。

之后重启vscode，即可完成汉化。

运行插件

搜索C/C++，并安装如图插件。

搜索Code Runner，并安装如图插件。
